Simulador de Fondo de Acumulación y Retiro

Esta herramienta te permite planificar financieramente el crecimiento y sostenibilidad de tu fondo de inversión, simulando escenarios de acumulación y retiro con parámetros personalizados.

📈

Acumulación

Simula el crecimiento de tu fondo con aportaciones anuales e interés compuesto. Visualiza cómo crece tu patrimonio año a año.

Saldo Final = (Saldo Inicial + Aportación) × (1 + Tasa)
💸

Retiros

Analiza los escenarios de retiro considerando múltiples fuentes de inversión, tasas de cambio e inflación.

Retiro Anual = Retiro₁ × (1 + Inflación)^(Año-1)
📊

Saldo del Fondo

Visualiza la evolución del fondo durante la fase de retiro, incluyendo poder adquisitivo real y cobertura en años.

Saldo Final = Saldo Inicial + Intereses − Retiros

💡 Cómo usar este simulador

  • Editable Los campos resaltados en azul son modificables.
  • Auto-cálculo Los resultados se actualizan automáticamente al cambiar parámetros.
  • Fórmulas Haz clic en "Fórmulas" para ver las ecuaciones utilizadas.
  • Guardar Guarda múltiples escenarios en la nube con Supabase.

🧮 Fórmulas Utilizadas

📈 Acumulación

Intereses Ganados
In = Sinicial × r
El interés se calcula sobre el saldo al inicio del período, antes de sumar la aportación del año.
Saldo Final del Año
Sfinal = Sinicial + Aportación + In
El saldo al final es la suma del saldo inicial, la aportación anual y los intereses generados.
Aportación Acumulada
Aacum = Σ Aportacióni (i = 1..n)
Suma total de dinero aportado desde el año 1 hasta el año n, excluyendo el monto inicial.

💸 Retiros

Requerimiento Anual (dólares)
RUSD = (Reqmensual × 12) × TC
El requerimiento en euros × 12 meses, convertido a dólares con la tasa de cambio.
Retiro con Inflación (año n)
Rn = R1 × (1 + π)n-1
El retiro crece cada año según la tasa de inflación π, compuesto año a año.
Tasa Ponderada Mixta
rpond = (ELSAL × rELSAL + Ext × rExt) / Fondo Total
Media ponderada de las tasas según el peso relativo de cada inversión en el fondo total.
Ingreso Anual (por escenario)
Ingreso = Fondo × Tasa
El ingreso que generaría el fondo si todo estuviera en ELSAL, en el extranjero, o mixto.

📊 Saldo del Fondo

Intereses del Período
In = Sinicial,n × rpond
Los intereses se calculan sobre el saldo inicial del año, usando la tasa ponderada mixta.
Saldo Final
Sfinal,n = Sinicial,n + In − Rn
Al saldo inicial se le suman los intereses y se restan los retiros del período.
Poder Adquisitivo
PAn = Sfinal,n / (1 + π)n
El saldo deflactado a valor presente para reflejar el poder adquisitivo real.
Cobertura en Años
Cn = Sfinal,n / Rn
Cuántos años de retiro quedan cubiertos con el saldo actual del fondo.
Ingresos − Retiros
Δn = In − Rn
Si es positivo, el fondo genera más de lo que se retira. Si es negativo, se consume capital.

⚙️ Configuración de Supabase

Ingresa las credenciales de tu proyecto Supabase para guardar y cargar escenarios en la nube.

Ejecuta este SQL en tu proyecto Supabase (SQL Editor):

CREATE TABLE IF NOT EXISTS financial_simulations (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  created_at  TIMESTAMPTZ DEFAULT NOW(),
  updated_at  TIMESTAMPTZ DEFAULT NOW(),
  name        TEXT NOT NULL,
  params      JSONB NOT NULL,
  user_email  TEXT DEFAULT 'default'
);

-- Habilitar Row Level Security (opcional)
ALTER TABLE financial_simulations ENABLE ROW LEVEL SECURITY;
